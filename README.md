# Co-Writer System Plugins

Official plugin marketplace for [The Co-Writer System](https://app.alexmcfarland.ai).

## Available Plugins

| Plugin | Description |
|--------|-------------|
| `youtube` | Write scripts, descriptions, companion newsletters, and titles |

## Install

Add this marketplace in Claude Cowork or Claude Code:

```
/plugin marketplace add alexmcfarland-ai/co-writer-plugins
```

Then install individual plugins:

```
/plugin install youtube@co-writer-system
```
