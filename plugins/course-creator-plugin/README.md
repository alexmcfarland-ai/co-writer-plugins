# Course Creator Plugin

Build and launch online courses — structure your curriculum, write lessons, create sales pages, and plan your launch sequence.

## Skills

| Skill | Description |
|-------|-------------|
| **course-outline** | Full course structure with modules, lessons, learning objectives, and time estimates |
| **lesson-writer** | Write individual lessons with objectives, content, examples, takeaways, and exercises |
| **course-sales-page** | Course-specific sales page with curriculum breakdown, instructor bio, FAQ, and pricing |
| **course-launch-sequence** | 10-email launch sequence: warmup → cart open → proof → urgency → final call |
| **module-intro** | Write module introductions connecting what's coming to what came before (200-400 words) |

## Context Files

All skills read these files if they exist:
- **USER.md** — who you are
- **BUSINESS.md** — your business and positioning
- **ICP.md** — your ideal customer
- Any installed voice/brand voice skill
