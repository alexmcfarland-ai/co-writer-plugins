# YouTube Plugin

Complete YouTube content creation suite for the Co-Writer System.

## Commands

| Command | Description |
|---------|-------------|
| `/youtube:script [topic]` | Write a full YouTube script with hooks, timed sections, end card, and title suggestions |
| `/youtube:description [title]` | Write an SEO-optimized video description |
| `/youtube:companion [transcript]` | Write a Substack companion newsletter for a video |
| `/youtube:titles [topic]` | Generate click-worthy title options with a recommended top 3 |

## Setup

This plugin works best with context files in your Cowork project folder:
- **USER.md** — who you are, how you work
- **BUSINESS.md** — your business, brand voice, content strategy
- **ICP.md** — your ideal audience

If these files exist, every command reads them automatically before producing output.

## Voice

If you have a voice skill installed in Claude Desktop, it will be loaded automatically. No extra setup needed.
