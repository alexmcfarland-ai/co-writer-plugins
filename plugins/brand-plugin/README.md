# Brand Plugin

Optimize your professional presence — social bios, LinkedIn profile, brand audits, and elevator pitches.

## Skills

| Skill | Description |
|-------|-------------|
| **social-bio-generator** | Platform-optimized bios (3-5 options each) for LinkedIn, X, and Instagram |
| **linkedin-optimizer** | Comprehensive 8-step LinkedIn profile optimization plan |
| **brand-audit** | Score content against your brand voice — voice consistency, ICP alignment, tone match (1-10) |
| **elevator-pitch** | 30-second pitch in 3 variations: casual, professional, and written (max 50 words each) |

## Context Files

All skills read these files if they exist:
- **USER.md** — who you are
- **BUSINESS.md** — your business and positioning
- **ICP.md** — your ideal customer
- Any installed voice/brand voice skill
