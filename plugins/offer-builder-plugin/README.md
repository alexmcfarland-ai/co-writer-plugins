# Offer Builder Plugin

Build irresistible offers, sales pages, email sequences, and lead magnets for digital products and services.

## Skills

| Skill | Description |
|-------|-------------|
| **sales-email-sequence** | Strategic 20-email launch sequence balancing education and sales |
| **sales-page** | Complete sales page with hero, pain, solution, features, FAQ, pricing, and CTA |
| **lead-magnet** | Design a lead magnet: format recommendation, content outline, and landing page copy |
| **value-stack** | Build a value stack table with deliverables, bonuses, values, and pricing |

## Commands

| Command | Description |
|---------|-------------|
| `/offer-builder:offer` | Guided 7-step workflow to build a complete irresistible offer from scratch |

## Context Files

All skills read these files if they exist:
- **USER.md** — who you are
- **BUSINESS.md** — your business and positioning
- **ICP.md** — your ideal customer
- Any installed voice/brand voice skill
