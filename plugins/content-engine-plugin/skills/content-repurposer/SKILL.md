# Content Repurposer

**Purpose:** Take one piece of content and transform it into platform-native pieces for YouTube, newsletter, Twitter, and LinkedIn — plus 5 alternative titles. Each output adapts the core insight for the platform's audience, format, and engagement patterns.

## When to Activate

Use this skill when:
- User provides content and wants it repurposed
- User mentions "turn this into", "repurpose", or "adapt for other platforms"
- User wants to maximize the reach of one piece of content

## Core Principle

**Repurpose ≠ copy-paste.** Each platform has different audience intent, format constraints, and engagement patterns. The core insight stays the same; the expression changes completely.

## Process

### Step 1: Identify the Core Insight

From the source content, extract:
- **The one big idea** (in one sentence)
- **Supporting points** (3-5 sub-ideas)
- **Best example or story** (the most compelling proof point)
- **The action item** (what should the audience do?)

### Step 2: Produce 5 Outputs

## Output Format

```markdown
# Content Repurposer: [Source Title]

**Core insight:** [One sentence]
**Source format:** [Article / Video / Newsletter / etc.]

---

## 1. YouTube Script Outline

**Title:** [Compelling YouTube title]
**Hook (first 30 seconds):**
[What to say to keep viewers watching — specific opening lines]

**Structure:**
1. [Section 1 — topic + key point + estimated time]
2. [Section 2]
3. [Section 3]
4. [Section 4]
5. [Conclusion + CTA]

**Estimated length:** [X] minutes
**Thumbnail concept:** [Brief description of what the thumbnail should communicate]

---

## 2. Newsletter Section

**Subject line:** [Email subject]
**Hook paragraph:** [Opening 2-3 sentences that would work in an email]

**Key sections:**
- [Section header 1 + brief description]
- [Section header 2 + brief description]
- [Section header 3 + brief description]

**Closing CTA:** [What action to prompt]
**Estimated word count:** [800-1200]

---

## 3. Twitter/X Thread

**Tweet 1 (Hook):** [Opening tweet — under 280 chars] 🧵
**Tweet 2:** [Point 1 — under 280 chars]
**Tweet 3:** [Point 2 — under 280 chars]
**Tweet 4:** [Point 3 — under 280 chars]
**Tweet 5:** [Example/story — under 280 chars]
**Tweet 6:** [Key takeaway — under 280 chars]
**Tweet 7:** [CTA — under 280 chars]

Character counts included for each tweet.

---

## 4. LinkedIn Post

[Full LinkedIn post — 300-500 words]
[Hook above the fold]
[Story or observation]
[3-5 insights]
[CTA]

**Word count:** [X] words

---

## 5. Title Options

Five alternative titles for the core content, each with a different angle:

1. **[Results-first]:** [Title]
2. **[How-to]:** [Title]
3. **[Contrarian]:** [Title]
4. **[Curiosity gap]:** [Title]
5. **[List-based]:** [Title]

**Recommended for each platform:**
- YouTube: Title #[X]
- Newsletter: Title #[X]
- Twitter thread: Title #[X]
- LinkedIn: Title #[X]
```

## Quality Checklist

- [ ] Core insight clearly identified in one sentence
- [ ] YouTube outline includes hook, structure, timing, and thumbnail concept
- [ ] Newsletter section has subject line, sections, and word count estimate
- [ ] Twitter thread has 6-8 tweets with character counts, all under 280
- [ ] LinkedIn post is 300-500 words with hook above fold
- [ ] 5 title options with different angles
- [ ] Each platform version feels native (not copy-pasted)
- [ ] All versions share the same core insight
- [ ] Voice matches context profiles across all outputs
