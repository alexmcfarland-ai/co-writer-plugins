# Newsletter Plugin

Write authority newsletters, how-to guides, case studies, and more — each skill produces a specific newsletter format ready to publish.

## Skills

| Skill | Description |
|-------|-------------|
| **thought-leadership** | 800-1500 word authority newsletter with subject lines, skimmable headers, and actionable content |
| **how-to-guide** | Step-by-step tutorial newsletter from any input (transcripts, notes, outlines) |
| **contrarian-take** | Newsletter built around a strong opinion that challenges conventional wisdom (800-1200 words) |
| **case-study** | Newsletter breaking down a real result — Situation → Action → Results → Lesson (600-1000 words) |
| **listicle-newsletter** | "7 things I learned" / "5 mistakes" format with bold takeaways (600-1000 words) |
| **story-lesson** | Personal story → lesson → actionable takeaway format (800-1200 words) |
| **newsletter-repurposer** | Takes 1 newsletter, produces 5 new angles with titles, hooks, and talking points |

## Context Files

All skills read these files if they exist:
- **USER.md** — who you are
- **BUSINESS.md** — your business and positioning
- **ICP.md** — your ideal customer
- Any installed voice/brand voice skill
