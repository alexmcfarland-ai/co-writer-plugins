# Ghostwriting Plugin

Set up a complete co-writer system for your ghostwriting clients — build their context profiles, capture their voice, map their intellectual property, and plan their content.

## Skills

| Skill | Description |
|-------|-------------|
| **voice-capture** | Analyze writing samples to create a complete voice profile skill file (.md) |
| **context-builder** | Build BUSINESS.md and ICP.md for a client from raw info, notes, or transcripts |
| **ip-mapper** | Map all client IP: stories, opinions, frameworks, phrases, and topic areas |
| **content-map** | Turn an IP map into a 4-week content plan with topics, angles, and formats |

## Commands

| Command | Description |
|---------|-------------|
| `/ghostwriting:setup` | Guided workflow to build a client's complete co-writer system step by step |
| `/ghostwriting:interview` | Generate a voice capture questionnaire (15-20 questions) to send to a client |

## Context Files

All skills read these files if they exist:
- **USER.md** — who you are
- **BUSINESS.md** — your business and positioning
- **ICP.md** — your ideal customer
- Any installed voice/brand voice skill
