# Freelance Plugin

Win and manage client work — proposals, scopes of work, progress reports, cold pitches, and case studies.

## Skills

| Skill | Description |
|-------|-------------|
| **client-proposal** | Professional proposal with exec summary, approach, scope, timeline, and pricing (2-4 pages) |
| **scope-of-work** | Detailed SOW with deliverables, timeline, revision policy, payment terms, and signatures |
| **client-report** | Progress/results report with metrics, challenges, next steps, and action items |
| **cold-pitch** | Cold outreach email tied to a specific trigger — under 150 words, 3 versions + follow-up |
| **case-study-doc** | Professional case study: background, challenge, solution, results, takeaways (500-800 words) |

## Context Files

All skills read these files if they exist:
- **USER.md** — who you are
- **BUSINESS.md** — your business and positioning
- **ICP.md** — your ideal customer
- Any installed voice/brand voice skill
