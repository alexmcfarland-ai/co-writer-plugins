# Social Plugin

Create high-performing social content — threads, LinkedIn posts, Substack notes, batched content, and endless content ideas from a single topic.

## Skills

| Skill | Description |
|-------|-------------|
| **substack-note** | Short-form Substack notes using 10 proven structural patterns and psychology |
| **twitter-thread** | 8-12 tweet threads with hook, body, and CTA — includes character counts |
| **linkedin-post** | Long-form LinkedIn posts (300-600 words) with hook, story, insights, and CTA |
| **instagram-caption** | Short, punchy captions with hook, value, CTA, and hashtag suggestions |
| **social-batch** | One topic → full suite: thread + LinkedIn + Instagram + Substack note + 5 tweets |
| **content-ideas** | 20 content ideas from 1 topic using 10 proven content lenses |

## Context Files

All skills read these files if they exist:
- **USER.md** — who you are
- **BUSINESS.md** — your business and positioning
- **ICP.md** — your ideal customer
- Any installed voice/brand voice skill
