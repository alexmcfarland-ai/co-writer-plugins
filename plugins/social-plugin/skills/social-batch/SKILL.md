# Social Content Batch

**Purpose:** Take one topic, idea, or piece of content and produce a complete multi-platform content suite in a single pass. Each piece shares the same core message but is natively adapted for its platform.

## When to Activate

Use this skill when:
- User wants to batch content creation from one topic
- User mentions "content suite", "multi-platform", or "repurpose for social"
- User provides a topic and wants it adapted across platforms
- User wants to maximize output from one idea

## Core Principle: Adapt, Don't Copy-Paste

Each platform has different:
- **Audience intent** (LinkedIn = professional growth, X = ideas + personality, Instagram = visual + lifestyle, Substack = depth + relationship)
- **Format constraints** (character limits, visual requirements, reading context)
- **Engagement patterns** (what gets shared, saved, commented on)

The same idea must be RE-EXPRESSED for each platform, not copied across them.

## Output Components

### 1. Twitter/X Thread (8-10 tweets)

**Format:**
- Hook tweet with 🧵 emoji
- One idea per tweet, under 280 chars each
- CTA as final tweet
- Include character counts

**Tone:** Pithy, sharp, personality-driven. Compressed wisdom.

### 2. LinkedIn Post (300-500 words)

**Format:**
- Hook line (above the "...see more" fold)
- Story or observation
- 3-5 numbered insights
- CTA

**Tone:** Professional but human. Story-driven. No corporate speak.

### 3. Instagram Caption (Under 150 words)

**Format:**
- Hook line (before truncation)
- 2-3 sentences of value
- CTA
- 5-15 hashtags

**Tone:** Casual, personality-forward, visual-mindset.

### 4. Substack Note (3-7 sentences)

**Format:**
- Choose the best note type for this topic (see formulas below)
- Apply structural pattern
- Keep under 7 sentences

**Note types:** Single-punch wisdom, pattern observation, contrarian statement, problem→solution, list-based tactical, or direct advice.

### 5. Five Standalone Tweets (280 chars each)

**Each tweet is a complete thought** that can be posted independently throughout the week. Push the topic through different lenses:

1. **The bold claim** — State the core idea with conviction
2. **The example** — A specific instance that illustrates the point
3. **The question** — Turn the idea into a thought-provoking question
4. **The contrarian angle** — Challenge the conventional take on this topic
5. **The actionable tip** — One specific thing the reader can do today

Include character counts for each.

## Quality Checklist

- [ ] All pieces share the same core message
- [ ] Each piece feels native to its platform (not copy-pasted)
- [ ] Twitter thread: 8-10 tweets, char counts, hook + CTA
- [ ] LinkedIn post: 300-500 words, hook above fold, no corporate speak
- [ ] Instagram caption: Under 150 words, hashtags, visual-mindset
- [ ] Substack note: 3-7 sentences, right structural formula
- [ ] 5 standalone tweets: Different lenses, all under 280 chars
- [ ] Voice consistent across all platforms (adapted for tone, not changed)
- [ ] No hardcoded handles/links — pulled from context files
- [ ] Posting schedule provided
