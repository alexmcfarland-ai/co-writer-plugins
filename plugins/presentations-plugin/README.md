# Presentations Plugin

Create polished slide decks — presentation builder, pitch decks, and workshop slides.

## Skills

| Skill | Description |
|-------|-------------|
| **presentation-builder** | Polished HTML slide deck with scroll-snap navigation, animations, and distinctive design |
| **pitch-deck** | Business/product pitch: Problem → Solution → Market → Product → Model → Traction → Ask (10-12 slides) |
| **workshop-slides** | Teaching deck for 60-90 min sessions with concept slides, exercises, and facilitator guide |

## Context Files

All skills read these files if they exist:
- **USER.md** — who you are
- **BUSINESS.md** — your business and positioning
- **ICP.md** — your ideal customer
- Any installed voice/brand voice skill
